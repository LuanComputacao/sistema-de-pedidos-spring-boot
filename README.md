# sistema de pedidos

# Frameworks

* BackEnd
  * spring-boot
* FrontEnd
  * bootstrap

# Tecnologias

* Hibernate
* JavaScript
  * JQuery 
* JSP
  * jstl
* REST API
* TomCat jasper

# Como executar a aplição
`mvn spring-boot:run`

# Como acessar a aplicação
`localhost:12000`
